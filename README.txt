1024 - Warpcast Mini-App (NETLIFY-READY)

This folder contains the built files ready for deployment to Netlify.

To deploy:
1. Go to https://app.netlify.com/
2. Click "Add new site" > "Deploy manually"
3. Drag and drop this entire folder onto the Netlify dashboard
4. Wait for deployment to complete
5. Your mini-app will be available at the Netlify URL (e.g., https://your-site-name.netlify.app)

After deployment:
1. Return to the Warpcast Mini-App Setup tool
2. Enter your actual deployment URLs (the Netlify URL where your app is deployed)
3. Generate the final files with the correct URLs

Important: Make sure your .well-known/farcaster.json file is accessible at your domain.
